A macOS menu bar app that turns your desktop into a beautiful calendar wallpaper.

Download for macOS View on GitHub

Features

🎨

10 Built-in Themes

Catppuccin, Nord, Dracula, Gruvbox, Rosé Pine, and more. Or create your own.

✍️

Handwriting Fonts

Elegant month display with Snell Roundhand, Zapfino, Bradley Hand, and other script fonts.

📊

Progress Tracker

Past days highlighted with a different color — your calendar becomes a visual progress bar.

📅

Event Integration

Today's events from Apple, Google, and Outlook calendars shown right on the wallpaper.

🖥️

Per-Display Settings

Different themes and layouts for each connected monitor. Retina-ready rendering.

🔄

Auto-Update

Wallpaper refreshes at midnight, on wake, and when displays change. App updates via Sparkle.
